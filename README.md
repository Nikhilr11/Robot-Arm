# Robot-Arm
This is a 5 Axis Robotic Arm controlled via a glove by having a potentiometer at each finger controlling each motor.
It can also be controlled using a game controller. Uploaded are the code files for the glove and the game contoller.

Parts:
4 270 degree servo motors
1 180 degree micro servo
3d printed arms
3d printed claw
3d printed glove
Esp-32 Dev Board
Breakout Board
5 Potentiometers
